# JavaScript Learning Project

## Overview
This is a simple static HTML/JavaScript learning project that demonstrates various JavaScript concepts including:
- Primitive values vs reference values
- Object manipulation
- Functions and arrow functions

## Project Structure
- `index.html` - Main HTML file
- `js/` - JavaScript files directory
  - `valor-referencia.js` - Currently active script demonstrating value vs reference behavior

## Running the Project
The project runs as a static website served on port 5000 using Python's http.server.

## Deployment
Configured for static deployment - the entire project directory is served as static files.
